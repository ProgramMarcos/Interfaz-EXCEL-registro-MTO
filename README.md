Interfaz creada para el registro de intervenciones de mantenimiento y la recopilación de datos.

Esto es un ejemplo de interfaz -- cada situación requerirá de una interfaz personalizada adaptada a partir de ésta.

La interfaz se ha realizado con VBA de EXCEL, permitiendo su integración en cualquier planta industrial y sin el 
requerimiento de programas o librerías externas que puedan generar conflictos de ciberrseguridad. Sólo se necesita 
EXCEL y una licencia de Microsoft.
